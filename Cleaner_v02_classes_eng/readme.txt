Инструкция по использованию программы KeyWord Cleaner

Программа написана на Python 3.10. Программа предназначена для чистки списков ключевых фраз от строк содержащих вхождения из файла со списком минус слов, а также осуществляет очистку спец.символов и лишних пробелов в оставшихся строках. 
Данная инструкция поможет вам понять, как использовать программу.



Описание программы:

Программа состоит из трех классов и функции main(), которая является точкой входа.

DatabaseManager: Класс для управления операциями с базой данных SQLite. Включает функции для создания таблицы, вставки фраз, удаления фраз с минус словами и экспорта данных в CSV файл.

FileManager: Класс для операций с файлами, включая загрузку конфигурации, обработку файлов с поисковыми фразами и минус словами, а также очистку текста.

MenuManager: Класс для отображения и обработки меню.

main(): Основная функция, объединяющая все классы и обеспечивающая интерфейс взаимодействия с пользователем через консольное меню.



Инструкция по использованию:

Для корректной работы программы убедитесь, что у вас установлены следующие модули:

Python (рекомендуемая версия 3.6+)
Модули: os, datetime, sqlite3, re, pandas, tqdm, csv, json
Если какой-либо библиотеки нет, вы можете установить их с помощью команды:

pip install sqlite3
pip install pandas
pip install tqdm



Настройка конфигурации:

Вам необходимо создать файл config.json в той же папке, где находится программа. В файле config.json укажите следующую информацию:

"log": Параметры для логирования, включая имя файла.
"database": Параметры для работы с базой данных, включая имя файла.
Пример config.json:

json
Copy code
{
    "log": {
        "filename": "app_log.log"
    },
    "database": {
        "filename": "search_phrases.db"
    }
}


Запуск программы:

Запустите программу, выполните следующие шаги:

1 Запустите файл main.py
2 Программа начнет работу и отобразит главное меню.
3 Введите букву операции (a/b/c) и нажмите Enter.

Операции:

a. Экспорт данных в CSV файл:
Введите начальную дату в формате dd-mm-yyyy.
Введите конечную дату в формате dd-mm-yyyy.
Программа экспортирует данные из базы данных в CSV файл в указанном диапазоне дат.

b. Добавление новых поисковых фраз:
Введите путь к CSV файлу с поисковыми фразами.
Программа обработает файл, очистит фразы и добавит их в базу данных.
Введите путь к CSV файлу с негативными словами.
Программа удалит фразы, содержащие негативные слова, из базы данных.

c. Выход:
Программа завершит свою работу.
Логирование:

Все действия программы будут заноситься в лог-файл, указанный в конфигурации.



Важно:

Все даты должны быть в формате dd-mm-yyyy.
CSV файлы должны быть в кодировке UTF-8.
Убедитесь, что файлы с поисковыми фразами и негативными словами имеют правильный формат.
